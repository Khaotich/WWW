Jquery działa na zasadzie łapania selektora i nadawaniu mu różnych działań zdefiniowwanych w funkcjach jquery.

$(selektor).funkcja(argumenty);

Selektor to elementy na stronie: td, #=id, .=class, które są podawane działaniom funkcji. Jeśli nie podamy żadnego agrumentu funkcji to zostanie zwrócony atrybut lecz jeśli podamy to coś ustawimy.

A

  • Bez argumentów
    • css - zwróci właściwości CSS podanego elementu
    • html - zwróci innerHTML pierwszego dopasowanego elementu
    • text - zwraca treść tekstową wszystkich dopasowanych elementów
    • attr - zwraca wartość podanego atrybutu dopasowanego elementu
    • val - zwraca wartość atrybutu value dopasowanego elementu
  • Z argumentami
    • css - ustawi podane właściwości CSS podanego elementu
    • html - dodaje innerHTML do pierwszego dopasowanego elementu
    • text - dodaje tekst do wszystkich dopasowanych elementów
    • attr - dodaje wartości do podanych argumentów dopasowanego elementu
    • val - ustawia wartośc atrybutu value dopasowanego elementu

B

  • hide - ukrywa podane elementy strony (podobne do css display: none;)
  • remove - usuwa podany element z jego podrzędnymi
  • visible - selektor wybierający wszystkie widoczne i dopasowane elementy
  • slideUp - przesuwa w górę element ukrywając inny
  • fadeOut - stopniowo zmienia krycie wybranych elementów, efekt blaknięcia

C

  • click - $(selektor).click(function(){})
  • mouseover - $(selektor).mouseover(function(){})
  • mouseout - $(selektor).mouseout(function(){})
  • keydown - $(selektor).keydown(function(){})

D

  • before - wstawia określoną treść przed wybranymi elementami
  • after - wstawia określoną treść po wybranych elementach
  • prepend - wstawia określoną treść na początku wybranych elementach
  • append - wstawia określoną treść na końcu wybranych elementach
  • remove - usuwa określoną treść wybranych elementów
  • empty - usuwa elementy potomne wybranego elementu

E

animate - wykona niestandardową animację zestawu właściwości CSS
